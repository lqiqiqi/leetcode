# Q
Invert a binary tree.

Example:
```
Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9
Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```

# My Ans
```
class Solution:
    def invertTree(self, root):
        """
        :type root: TreeNode
        :rtype: TreeNode
        """
        if not root:
            return None
        
        if root.left or root.right:
            temp = root.left
            root.left = root.right
            root.right = temp
        self.invertTree(root.left)
        self.invertTree(root.right)
        
        return root
```
